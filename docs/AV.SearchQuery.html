AV. SearchQuery

new SearchQuery()

Source:
  • search.js, line 114
Since:
  • 0.5.1
App searching query.Use just like AV.Query: Visit App Searching Guide for more details.
Example
var query = new AV.SearchQuery('Player');
  query.queryString('*');
  query.find().then(function(results) {
    console.log('Found %d objects', query.hits());
    //Process results
  });
